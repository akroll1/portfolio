Featured Project

Previous Next

This Lego(IoT) house may look like a toy, but don't be fooled! I created this project from scratch and I'm very proud of it. My Lego(IoT) project is a Lego house that uses an Arduino board, which is an open-source micro-controller, to control and receive data through the browser from numerous sensors, lights and motors in the Lego(IoT) house. I used Node.js as my server and I used both web sockets and PubNubs network to send messages globally. The project can stream data from the house to plot.ly, an online analytics and data visualization tool, where it can be referenced in real time. This data, and the unique ways it can be visualized, can help find and solve problems and issues that we might not recognize as being problems yet.

You may notice the pill bottle in the background. One feature of the Lego(IoT) house is that I've created a service to solve a real problem faced by people everyday- that of taking medications on time. This service sends real reminders to people, via PubNub's network, to take their medications at the appointed time and also records the time it was taken, and that information can be analyzed in a couple of different ways. Fun, yes. Toy, no.
